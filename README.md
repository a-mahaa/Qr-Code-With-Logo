# Qr-Code-With-Logo

Description:
This project is a QR code generator built with Python. The program creates a QR code with a custom logo image placed in the center of the QR code.

Language Used:
Python

Libraries Used:
qrcode
PIL (Pillow)

Additional Files Required:
This project requires an external image file such as "csumb.jpg" to use as the logo inside the QR code.

Features:
- Generate QR codes
- Add custom logo to QR code
- Resize and place logo automatically
- Save QR code as image file
- Custom QR code colors
